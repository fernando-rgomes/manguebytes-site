# 🌱 Manguebytes - Landing Page

Landing page oficial da **Manguebytes**, desenvolvida em **React + Vite** para garantir performance, modularidade e responsividade.

---

## 🚀 Tecnologias utilizadas
- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/) (bundler e dev server)
- [CSS3](https://developer.mozilla.org/docs/Web/CSS)
- [Node.js](https://nodejs.org/)

---

## 📂 Estrutura do projeto
manguebytes-site/
├── public/              # Arquivos públicos (favicon, index.html, imagens estáticas)
├── src/
│   ├── assets/          # Imagens e ícones
│   ├── components/      # Componentes reutilizáveis
│   ├── App.jsx          # Componente principal
│   └── main.jsx         # Ponto de entrada do Vite
└── package.json         # Dependências e scripts


---

## ⚙️ Como rodar o projeto

### 1. Clonar o repositório
git clone https://github.com/fernando-rgomes/manguebytes-site.git

### 2.Instalar dependências
npm install

### 3. Rodar em ambiente local
npm run dev
